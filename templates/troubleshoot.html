CUBY.AI Troubleshoot

Troubleshooting Guide for CUBY.AI

CUBY not responding:

  1. Ensure microphone is enabled and properly connected.
  2. Check internet connection if required for specific commands.
  3. Restart CUBY or your device.

Misunderstanding commands:

  1. Speak clearly and avoid background noise.
  2. Try rephrasing your command more simply.
  3. Check if the command requires specific keywords or formatting.

Time/Date (Commands 2 & 3):

  1. If providing incorrect information, check if CUBY uses your local time settings or needs manual adjustment.

Questions (Command 4):

  1. If CUBY can't define the answer, specify the context or source of the question.
  2. Ensure your question is complete and well-formed.

Notepad Functions (Commands 5-9 & 29):

  1. If notes aren't saving, check file permissions or storage space.
  2. Ensure "Writer" command is used for manual typing input.

Web Search (Command 10):

  1. If search results are inaccurate, refine your query or check your default browser settings.

Computer Control (Commands 11-13):

  1. Make sure you have administrator privileges to execute these actions.
  2. Ensure confirmation prompts are acknowledged for successful execution.

Music Playback (Commands 14 & 15):

  1. Verify if your songs are accessible and compatible with the media player.
  2. Double-check song names or try searching by artist/album for better accuracy.

Remember/Recall (Commands 16 & 17):

  1. Make sure you used the "Remember that" command properly beforehand.
  2. Use specific keywords or context when asking for remembered commands.

Screenshot (Command 18):

  1. Check if screenshots are saved to the designated location.
  2. Ensure necessary permissions are granted for CUBY to access storage.

Symbol Art (Command 24):

  1. Verify the image format is compatible for conversion.
  2. Try adjusting image size or clarity for better conversion results.

Offline Mode (Command 33):

  1. Confirm if CUBY requires an internet connection for certain commands.
  2. Ensure offline functionality is properly enabled before using offline mode.
